# Sistem Manajemen Perpustakaan

Program ini dibuat menggunakan Python 3 dengan database SQLite.

## Fitur
- CRUD Data Buku
- CRUD Data Anggota
- Peminjaman Buku
- Pengembalian Buku

## Teknologi
- Python 3
- SQLite
- Tabulate

## Cara Menjalankan Program

1. Clone repository
   https://github.com/okamahagangga/ManagementPerpustakaan

2. Masuk folder project
    masuk ke file projek disini ada dua jenis file dalam bentuk python dan dalam bentuk file yg sudah di compile di dalam folder dist         yaitu file exe

3. Install library

Install library terlebih dahulu:

pip install tabulate

4. Jalankan program

    a. Jalankan file:
    main_gui.exe

    atau

    b. Jalankan melalui Python:
    python main_gui.py
