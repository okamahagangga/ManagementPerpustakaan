# Sistem Manajemen Perpustakaan

Program ini dibuat menggunakan Python 3 dengan database SQLite.

## Fitur
- CRUD Data Buku
- CRUD Data Anggota
- Peminjaman Buku
- Pengembalian Buku

## Teknologi
- Python 3
- SQLite
- Tabulate

## Cara Menjalankan Program

1. Clone repository

git clone https://github.com/username/sistem-perpustakaan-python.git


2. Masuk folder project

cd sistem-perpustakaan-python

3. Install library

Install library terlebih dahulu:

pip install tabulate

4. Jalankan program

    a. Jalankan file:
    main_gui.exe

    atau

    b. Jalankan melalui Python:
    python main_gui.py
